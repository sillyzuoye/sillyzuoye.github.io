与其买三百块钱的鼠标不如去买三百块钱的蛋糕
——迟到的二十岁与迟到的反思

无论如何，我希望把保持充足的睡眠看作我人生中最最最最重要的事情

背景：在家里上网课，虽然节约了很多时间，但是效率仍然不会高，且并不能抽出大量时间做自己喜欢的事情，终于，彻夜难寐。

问题

对剧本的警惕性不足

症状：

1. 对剧本的厌恶
2. 对无聊生活的难以下手
3. 莫名其妙的对剧本痴迷
4. 无法从中学得任何东西

造成问题：

1. 对任何剧本抱有厌恶态度。
2. 无聊的生活中对有趣（？）/对事件发生的渴望，将自己代入男主或者什么，对所描述的虚无缥缈的世界观与人际关系产生渴望，并受到意识形态的影响和渗入。
3. 会对剧本的结局产生期待或者渴望的心情，会一直将这个剧本看下去或者看完，耽误学习时间和睡眠时间。
4. 从中不会学习到任何东西，学不到分镜或者剧本的写作方法，不能学到人物的情感的表达方法，没有总结时间（真的没有吗？），也不能通过看剧来学习语言，外语剧的言语说话速度太快而不能适应。

解决方法：

1. 在学习剧本的写作之前不看一切剧本
2. 生活本就是枯燥无味的，应当学会适应，按犬儒主义来行动一段时间吧
2.5. 将感兴趣的视频或者剧本列入计划/稍后再看之间，让理性来看视频而不是让力比多看视频
3. 从哪点开的剧本？为什么剧本或者剧情或者切片会走到我的面前？
4. 
* 视频：对视频中的运镜和分镜保持警戒
* 画面：对作画或线条保持警戒
* 音乐：对背景音乐的配乐保持警戒
* 语言：先从基础的听力练起，且直接上生肉而不是看着熟肉的字母进行学习
* 总体：选个好的看吧

对短视频/贴吧的警惕性不足

症状：

1. 会痴迷于无止境的下一个视频，时间无止境的消磨
2. 知道是错误的，但是无法手指的刷新动作
3. 自我良心的谴责与困倦感、无力感的到来

造成问题：

1. 耽误睡眠时间与学习时间
2. 自我良心的不安和愧疚

解决方法：

1. 减少或者禁止在推荐项目停留，减少一切的个性化推荐。
2. 做什么事情应当是有意义的，有目的，而不是碎片的，随机的。
3. 应当在金子堆里找钻石而不是一大坨屎中希望能随机到一粒金子，那是恶心的，如同在垃圾/水课中仔细听课找到有用的知识一样——令我厌恶。

对游戏的尸位素餐

症状：

1. 玩的并不是自己喜欢的游戏
2. 玩得并不过瘾

造成问题：

1. 玩游戏的罪恶感
2. 不能学到任何东西

解决方法:

1. 只玩自己想玩的游戏,不要勉强或者强求别人去玩或者一起玩什么游戏	。
2. 如果在网络对战游戏中，那就需要花时间去训练，而不是靠无意识的游玩来浪费时间

自身的懒惰与连锁反应

症状：

1. 熬夜睡得晚起得晚或者起得早没有精力
2. 会懒得去刷anki，刷anki的过程中会感到疲倦和无意义，记忆能力低下，刷anki需要大把时间而放弃刷anki
3. 因为没有去做记忆的事情，对睡午觉会感到罪恶感，于是乎不睡午觉，思考能力与记忆能力大幅下降，下午也放弃刷anki接着开摆
4. 因为一整天没有刷anki，晚上的时间太短不够刷，于是乎放弃，罪恶感导致我在短视频与剧本中寻找虚假的救赎，然后接着熬夜晚睡，一个完整循环

造成问题：

无止尽的循环导致进度缓慢以及自我认同的坍塌

解决方法：

睡一觉好的，重新拟定计划，打破循环
